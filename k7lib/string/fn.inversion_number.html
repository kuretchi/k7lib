Calculates the inversion number.

This function uses its argument destructively.

Examples

let mut s = [3, 1, 4, 1, 5];
assert_eq!(inversion_number::<_, u32>(&mut s), 3);

Time complexity

$\Theta(n \log(n))$ where $n$ = s.len()
